# Pitch-Perfect
First portfolio project for the iOS Developer Nanodegree Program at Udacity

## App description
An app that records a message and plays the audio back through user-selected filters.

* UIKit
* Foundation
* AVFoundation (AVAudioEngine)

* MVC
* Delegation
* Target-Action
